◆ n_kafka_event_fdesc()

static const char * n_kafka_event_fdesc ( const N_KAFKA_EVENT *  event)
static

return a " (file: ...)" log fragment naming the source file(s) of an event

Parameters
eventthe event to describe
Returns
a never-NULL const C string to append after the opaque pointer in a log line. When the event carries its source file(s) (event_files_to_delete, set when the event is produced from a monitored directory) the fragment names them; for events with no source file (string events, single-file sends, received events) it is empty so no misleading "(file: ...)" is printed.
