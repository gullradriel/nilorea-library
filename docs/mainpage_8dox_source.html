11\section aboutlib About Nilorea Library
12
13C tools that are usable as a collection of .c .h or a shared library/dll :
14
15Logging helpers, strings, linked lists, hash tables, thread pools, networking, ...
16
17The name is coming from one of my first project, which I failed over the time. The unfinished game gave birth to a usable library. The opposite may happen one of these days.
18
19See the modules sections for more informations

22\section require Requirements
23
24If using graphical/audio modules:
25
26- Allegro 5.0.0 or better http://alleg.sf.net/ (if HAVE_ALLEGRO is used)
27
28Mainly for thread and network module:
29
30- The socket libs (winsock or socket)
31
32- Pthread lib (posix pthread for windows or linux)
33
34- OpenSSL libs (if HAVE_OPENSSL is used)
35
36- PCRE library for pattern matching in various places
37
38\section install Installation
39Just type make and use the library or put the right files in your LIB and INCLUDE directory, link them with your code and let's go !
40Don't forget to define FLAGS for parts you don't need, it can save you memory and compilation time.
